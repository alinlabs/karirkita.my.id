## TENTANG PROYEK
**KarirKita** adalah platform portal lowongan kerja yang menghubungkan pencari kerja dengan perusahaan di seluruh Indonesia. Website ini dikembangkan dengan teknologi modern untuk memberikan pengalaman pengguna yang cepat, responsif, dan mudah digunakan.

## TEKNOLOGI UTAMA
- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, Framer Motion
- **Backend/API:** Cloudflare Workers, D1 Database
- **State Management:** React Context API
- **Routing:** React Router DOM

## CARA MENJALANKAN APLIKASI (LOKAL)

**Prasyarat:** Pastikan Node.js sudah terinstal di komputer Anda.

1. **Install Dependensi:**
   Jalankan perintah berikut di terminal:
   ```bash
   npm install
   ```

2. **Konfigurasi Environment:**
   Salin file `.env.example` menjadi `.env` dan sesuaikan variabel lingkungan yang diperlukan (seperti `GEMINI_API_KEY` jika menggunakan fitur AI).

3. **Jalankan Aplikasi:**
   Mulai server pengembangan lokal:
   ```bash
   npm run dev
   ```
   Akses aplikasi di `http://localhost:5173` (atau port yang ditampilkan).
